Introduction

Generative models are a class of machine learning models designed to learn underlying distribution of data and generate new samples that resemble the original dataset. In this project, we learn two types of generative models: VAE (Variational Autoencoders) and GAN (Generative Adversarial Networks). The goal is to understand how these models learn to represent and generate data, compare their performance and behavior.

In Part 1, both a basic autoencoder and a VAE are implemented and trained on the MNIST dataset to analyze reconstruction quality and latent space representation. In Part 2, a DCGAN (Deep Convolutional GAN) is implemented using a Pokemon image dataset to generate realistic images from random noise. This project experiments are conducted by adjusting latent dimensions, activation functions, and other hyperparameters to observe their effects on training stability and output quality.

The project highlights the strengths and limitations of each model, showing how VAEs give more structured latent space for sampling, while GANs can generate more realistic outputs.

PART 1: Variational Autoencoder (VAE)

Task 1: Basic Autoencoder

A fully connected autoencoder was implemented with an encoder that compresses the input image into latent representation and a decoder that reconstructs the image from the latent representation. The model was trained on the MNIST dataset using different latent dimensions (2, 8, and 32).

Task 2: Variational Autoencoder (VAE)

A VAE was implemented where the encoder outputs mean and variance, forming a Gaussian latent space. The reparameterization trick is used to sample latent vectors, and the decoder reconstructs images from these samples.

The loss includes reconstruction loss and KL divergence, which forces the latent space to follow a standard normal distribution.

The VAE loss decreases over time, showing successful learning of both reconstruction and latent structure. Although loss values are higher than the autoencoder, this is expected due to KL divergence.

Sampling from the latent space produces smooth and meaningful digits. Compared to the autoencoder, VAE outputs are clearer and more structured because the latent space is continuous.

Latent Space Visualization

The latent space shows smooth transitions between digits in the range [-1, 1], while extreme values in [-5, 5] produce more distortion. This shows that VAE learns a structured and continuous latent space.
